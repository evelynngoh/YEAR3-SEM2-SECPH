From the Application Development course, valuable knowledge and practical experience in mobile and web application development using Flutter were gained. The course provided exposure to UI/UX design, frontend and backend integration, state management, and database handling while developing interactive applications. A major learning experience came from the group project, which involved developing a real-case Attendance Management System for TVETMARA. Through this project, practical understanding of system development processes such as requirements analysis, interface design, module integration, and testing was strengthened. In addition, problem-solving, teamwork, and communication skills were improved throughout the development process. The project also highlighted the importance of developing user-friendly and efficient systems that can support real organizational needs. Overall, the course successfully enhanced both technical and collaborative skills in application development while providing experience in handling a real-world case study.
